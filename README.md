# Digital Control Systems

![Simulink](https://img.shields.io/badge/Simulink-0076A8?style=for-the-badge&logo=mathworks&logoColor=white)

MATLAB/Simulink implementations of digital control techniques for a rotational electromechanical system.

| Folder | Topic |
|---|---|
| `HA01` | Quasi-continuous PI control — Betragsoptimum |
| `HA02` | Digital magnitude optimum — P-T1 & dead-time designs |
| `HA03` | State-space control — Dead-Beat, Binomial, Butterworth |
| `HA04` | Parameter identification — IV4 algorithm |
| `HA05` | Kalman filter — with and without disturbance state |
